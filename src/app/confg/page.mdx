# Configuration

FakeTool can be customized using a `faketool.config.js` file in the root of your project.

---

## Example configuration

```js filename="faketool.config.js" copy
export default {
  port: 4000,
  outputDir: 'dist',
  features: {
    hotReload: true,
    minify: true,
  },
  theme: 'dark',
};
```

## Available Options

- port (number, default: 3000)
  Port on which the dev server will run.

- outputDir (string, default: "dist")
  The directory where production builds are stored.

- features.hotReload (boolean, default: true)
  Enables hot reloading in development.

- theme (string, default: "light")
  Theme for your project UI ("light" or "dark").
