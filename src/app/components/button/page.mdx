# Button

CyWeb ships with a few built-in components to help you build faster.  
You can import them directly from `CyWeb/components`.

---

## Button

A simple button component.

```jsx copy
import { Button } from 'CyWeb/components';

export default function Example() {
  return <Button label="Click Me" onClick={() => alert('Hello from CyWeb!')} />;
}
```
