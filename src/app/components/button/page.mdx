# Button

FakeTool ships with a few built-in components to help you build faster.  
You can import them directly from `faketool/components`.

---

## Button

A simple button component.

```jsx copy
import { Button } from 'faketool/components';

export default function Example() {
  return (
    <Button label="Click Me" onClick={() => alert('Hello from FakeTool!')} />
  );
}
```
